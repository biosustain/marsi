marsi - Metabolite Analogues for Strain Improvement
===================================================

marsi is an open-source software to created to identify strain design
targets 

Dependencies
------------
* eigen3 >= 3.0
* OpenBabel >= 2.2.3
* RDKit >= 2016
* mongodb
* glpk
* cplex (optional)
* Cython >= 0.25
* numpy >= 1.11


Quick Start
-----------

1. Install the Dependencies
2. <code> pip install marsi </code>
